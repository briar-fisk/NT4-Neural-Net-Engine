1//The Current Active Node Scaffold for Trace Encoding (CAN) is the scaffold we use to iteratively encode traces.
2//For example, a network with an I/O depth of 5 and a single tier w/ one 5 legged node may look like this:
3// Memory_Tier[1]: [0] //The top node, represents one permutation of the input tier. Multiple Memory tiers.
4// The memory tiers start at [1] since the State tier is technically Memory tier [0].
5// State_Tier (0): [0] [1] [2] [3] [4] //This is the tier with the states.
6// Each [] represents a pointer to a node. We iterate from bottom to top, using get_State_Node,
7// and get_Upper_Tier_Node() to fill it out tier by tier.
8
9//--Not using a polymorphic base class here, different CANs may have wildly different features.
